## 🚀 추천 구현 기능 (우선순위)

### 🔥 높은 우선순위

#### 1. **댓글 복원 기능 완성** ⭐ (일관성 유지)
- **이유**: 게시글 복원 기능과 일관성 유지
- **작업**: 삭제된 댓글 목록 화면 + 복원 버튼
- **예상 작업 시간**: 1-2시간

#### 2. **검색 기능** ⭐⭐ (README에 명시되어 있음)
- **이유**: README에 언급되어 있지만 미구현
- **기능**:
  - 제목/내용 검색
  - 작성자 검색
  - 실시간 검색 필터링
- **예상 작업 시간**: 2-3시간

#### 3. **정렬 및 필터링** ⭐⭐
- **기능**:
  - 최신순 / 조회수순 / 댓글수순
  - 수정된 게시글만 보기
  - 내가 작성한 게시글만 보기
- **예상 작업 시간**: 1-2시간

### 📊 중간 우선순위

#### 4. **게시글 좋아요/추천 기능**
- **기능**:
  - 좋아요 버튼 (중복 방지)
  - 좋아요 수 표시
  - 인기도 계산 (조회수 + 댓글수 + 좋아요수)
- **예상 작업 시간**: 2-3시간

#### 5. **페이지네이션 (무한 스크롤)**
- **이유**: 게시글이 많아질 때 성능 이슈 대비
- **기능**: 
  - 초기 20개 로드
  - 스크롤 시 추가 로드
- **예상 작업 시간**: 2-3시간

#### 6. **게시글 이미지 업로드**
- **기능**:
  - Firebase Storage 연동
  - 이미지 선택 및 업로드
  - 이미지 미리보기
- **예상 작업 시간**: 3-4시간

### 🎨 낮은 우선순위 (UX 개선)

#### 7. **사용자 프로필**
- **기능**:
  - 프로필 사진
  - 작성 게시글 수
  - 작성 댓글 수
  - 프로필 편집
- **예상 작업 시간**: 3-4시간

#### 8. **알림 기능**
- **기능**:
  - 내 게시글에 댓글 달렸을 때
  - 내 댓글에 대댓글 달렸을 때
  - Firebase Cloud Messaging 연동
- **예상 작업 시간**: 4-5시간

#### 9. **게시글 공유 기능**
- **기능**:
  - 게시글 URL 공유
  - SNS 공유
- **예상 작업 시간**: 2-3시간

#### 10. **태그/카테고리**
- **기능**:
  - 게시글 태그 추가
  - 태그별 필터링
  - 인기 태그 표시
- **예상 작업 시간**: 4-5시간

#### 11. **멀티 뷰 모드** (이전 논의)
- **기능**:
  - 리스트 뷰 / 그리드 뷰 / 카드 스택 뷰
- **예상 작업 시간**: 5-6시간

#### 12. **통계 및 분석**
- **기능**:
  - 일별 게시글 수
  - 인기 게시글 랭킹
  - 활성 사용자 통계
- **예상 작업 시간**: 3-4시간

## 💡 빠르게 구현 가능한 작은 기능들 (Quick Wins)

### 1. **게시글 수정 시간 표시 개선**
- 현재: "수정 N회"만 표시
- 개선: "마지막 수정: YYYY.MM.DD HH:mm" 추가

### 2. **댓글 수 표시**
- 게시글 목록에 댓글 수 표시
- 아이콘: 💬 N개

### 3. **게시글 미리보기 길이 조절**
- 현재: 2줄만 표시
- 설정: 미리보기 줄 수 선택 가능

### 4. **빈 화면 개선**
- 게시글이 없을 때 더 나은 UI
- 삭제된 게시글이 없을 때 더 나은 UI

### 5. **로딩 상태 개선**
- 스켈레톤 UI
- 더 부드러운 전환 효과

### 6. **다크 모드**
- Material Design 3 다크 테마 지원
