## 📋 상세 촬영 가이드

### 1. 게시글 목록 화면 (⭐⭐⭐ 필수)
**촬영 방법:**
1. 앱 실행 후 로그인 완료
2. 게시글 목록 화면으로 이동
3. 샘플 데이터가 생성되어 있는지 확인 (없으면 생성)
4. 여러 게시글이 보이도록 스크롤
5. 스크린샷 촬영

**포함 요소:**
- ✅ 여러 게시글 항목
- ✅ 작성자 이름
- ✅ 날짜/시간
- ✅ 조회수
- ✅ 수정 표시 (수정된 게시글이 있다면)

### 2. 게시글 상세 + 댓글 화면 (⭐⭐⭐ 필수)
**촬영 방법:**
1. 게시글 목록에서 게시글 선택
2. 게시글 상세 화면으로 이동
3. 댓글 및 대댓글이 표시된 게시글 선택
4. 댓글이 충분히 보이도록 스크롤
5. 스크린샷 촬영

**포함 요소:**
- ✅ 게시글 제목 및 내용
- ✅ 작성자 정보
- ✅ 조회수
- ✅ 댓글 여러 개
- ✅ 대댓글 (가능하면)
- ✅ 댓글 작성란

### 3. 로그인/회원가입 화면 (⭐⭐ 권장)
**촬영 방법:**
1. 앱 첫 실행 또는 로그아웃 상태
2. 로그인 또는 회원가입 화면으로 이동
3. **스타일 선택**:
   - **옵션 A (권장)**: 입력 폼을 채워진 상태로 촬영
     - 이름: `테스트 사용자`
     - 이메일: `test@example.com`
     - 비밀번호: `password123` (자동 마스킹됨)
   - **옵션 B**: 빈 폼 상태로 촬영
4. 스크린샷 촬영

**포함 요소:**
- ✅ 로그인 입력 필드
- ✅ 회원가입 입력 필드
- ✅ 앱 이름/로고

### 4. 게시글 작성 화면 (⭐ 권장)
**촬영 방법:**
1. 게시글 목록 화면에서 "작성" 버튼 클릭
2. 게시글 작성 화면으로 이동
3. 제목과 내용을 입력한 상태 (또는 빈 상태)
4. 스크린샷 촬영

**포함 요소:**
- ✅ 제목 입력 필드
- ✅ 내용 입력 필드
- ✅ 작성/취소 버튼

### 5. 삭제된 게시글 목록 (⭐ 선택)
**촬영 방법:**
1. 삭제된 게시글이 있는 상태
2. 메뉴에서 "삭제된 게시글" 선택
3. 삭제된 게시글 목록 화면으로 이동
4. 스크린샷 촬영

**포함 요소:**
- ✅ 삭제된 게시글 목록
- ✅ 복원 버튼

## 💡 팁

1. **릴리즈 모드로 촬영**: `flutter run --release`
   - Debug 모드 배지 제거
   - 실제 출시 앱과 동일한 느낌

2. **일관성 유지**: 모든 스크린샷에서 동일한 스타일 사용
   - 채워진 폼 vs 빈 폼 선택 후 일관성 유지

3. **샘플 데이터**: 게시글과 댓글이 충분히 있는 상태로 촬영

4. **화질**: 고해상도로 촬영하여 앱 스토어 품질 요구사항 만족
