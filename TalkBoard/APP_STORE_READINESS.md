# 앱 스토어 출시 준비 체크리스트

## ✅ 현재 구현된 기능

### 핵심 기능
- ✅ **사용자 인증**: 이메일/비밀번호 로그인, 회원가입
- ✅ **게시글 CRUD**: 생성, 읽기, 수정, 삭제
- ✅ **댓글 시스템**: 댓글 작성, 수정, 삭제
- ✅ **대댓글**: 댓글에 대한 답글 기능
- ✅ **실시간 동기화**: StreamBuilder로 실시간 업데이트
- ✅ **조회수**: 게시글 조회수 실시간 업데이트
- ✅ **수정 히스토리**: 게시글/댓글 수정 이력 관리
- ✅ **소프트 삭제**: 게시글/댓글 소프트 삭제 및 복원
- ✅ **삭제된 게시글 관리**: 자신이 삭제한 게시글만 확인 가능
- ✅ **Provider 패턴**: 상태 관리 개선 완료

### 기술적 완성도

- ✅ 보안 규칙 설정 가능 (문서 제공)

### 2. 앱 스토어 메타데이터

#### Google Play Store
- [ ] **앱 이름**: 톡보드 (또는 원하는 이름)
- [ ] **짧은 설명**: 140자 이내
- [ ] **긴 설명**: 앱 기능 상세 설명 (4000자 이내)
- [ ] **앱 아이콘**: 512x512px PNG (투명 배경 불가)
- [ ] **피처 그래픽**: 1024x500px
- [ ] **스크린샷**: 최소 2개 (휴대폰: 16:9 또는 9:16 비율)
  - 최소 2개: 스마트폰
  - 권장 2개: 태블릿
- [ ] **카테고리**: 커뮤니케이션 또는 소셜
- [ ] **콘텐츠 등급**: 전체 이용가 (또는 사이트별)
- [ ] **개인정보처리방침 URL**: 필수
- [ ] **이메일 연락처**: 필수
- [ ] **웹사이트**: 선택 (권장)

#### Apple App Store
- [ ] **앱 이름**: 톡보드 (30자 이내)
- [ ] **서브타이틀**: 30자 이내
- [ ] **설명**: 앱 기능 설명
- [ ] **키워드**: 검색 키워드 (100자 이내, 쉼표 구분)
- [ ] **앱 아이콘**: 1024x1024px PNG (투명 배경 불가, 둥근 모서리 제거)
- [ ] **스크린샷**: 최소 1개 (6.5인치 iPhone: 1284x2778px)
  - 권장: 5.5인치 iPhone (1242x2208px)
  - 권장: 6.5인치 iPhone (1284x2778px)
  - iPad Pro: 선택
- [ ] **앱 미리보기 비디오**: 선택 (권장)
- [ ] **카테고리**: 소셜 네트워킹
- [ ] **연령 등급**: 4+ (또는 적절한 등급)
- [ ] **개인정보처리방침 URL**: 필수
- [ ] **연락처 정보**: 필수

### 3. 기술적 준비사항

#### Android
- [ ] **서명 키 생성**: 릴리즈 빌드용 (현재 Debug 키 사용 중)
  - 위치: `android/app/key.jks` (Git에 커밋하지 말 것!)
  - 방법: `RELEASE_CHECKLIST.md` 참고
- [ ] **ProGuard 설정**: 선택사항 (코드 난독화)
- [ ] **앱 버전 관리**: `pubspec.yaml`의 `version` 필드
- [ ] **앱 아이콘 커스터마이징**: 현재 기본 아이콘
  - 위치: `android/app/src/main/res/mipmap-*/ic_launcher.png`

#### iOS
- [ ] **Apple Developer 계정**: App Store 배포 필수 ($99/년)
- [ ] **Bundle ID 확인**: Firebase 설정과 일치
- [ ] **앱 아이콘 커스터마이징**: 현재 기본 아이콘
  - 위치: `ios/Runner/Assets.xcassets/AppIcon.appiconset/`
- [ ] **인증서 및 프로비저닝 프로파일**: Xcode에서 자동 생성 가능

### 4. Firebase 프로덕션 설정

#### 필수 확인사항
