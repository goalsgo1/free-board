# Feature Graphic (그래픽 이미지) 가이드

## 📋 Feature Graphic이란?

**Feature Graphic**은 Google Play 스토어에서 앱 목록 상단에 표시되는 배너 이미지입니다.

- **위치:** 앱 목록 상단, 스크린샷 위에 표시
- **용도:** 앱의 주요 기능이나 브랜딩을 한눈에 보여줌
- **크기:** **정확히 1024 x 500px** (가로세로 비율 2:1)

---

## ✅ 요구사항

- **해상도:** 정확히 **1024 x 500px**
- **형식:** PNG 또는 JPEG
- **최대 크기:** 15MB
- **가로세로 비율:** 2:1 (1024:500)

---

## 🎨 디자인 가이드

### 포함할 내용:

1. **앱 로고/이름**
   - "톡보드" 로고 또는 텍스트
   - 왼쪽 또는 중앙에 배치

2. **주요 기능 키워드**
   - "실시간 게시판"
   - "댓글 및 대댓글"
   - "소프트 삭제 및 복원"

3. **디자인 팁**
   - 깔끔하고 읽기 쉬운 디자인
   - 텍스트와 배경의 대비가 충분해야 함
   - 다양한 화면 크기에서 잘 보여야 함

### 디자인 도구:

- **Figma:** 무료, 온라인 디자인 도구
- **Canva:** 무료, 템플릿 제공
- **Photoshop:** 전문 디자인 도구
- **온라인 도구:** https://www.canva.com/

---

## 📝 현재 상황

**문제:**
- 현재 선택된 이미지: `01_login.png` (800x1244px)
- 요구사항: 1024x500px (정확히)
- 이미지 크기가 요구사항과 맞지 않음

---

## ✅ 해결 방법

### 옵션 1: 새로 디자인 (권장)

1. **디자인 도구 선택:**
   - Canva (https://www.canva.com/) - 무료, 쉬움
   - Figma (https://www.figma.com/) - 무료, 전문적

2. **템플릿 사용:**
   - Canva에서 "Google Play Feature Graphic" 템플릿 검색
   - 1024x500px 템플릿 선택

3. **디자인 요소 추가:**
   - 톡보드 로고/이름
   - 주요 기능 설명
   - 색상: 톡보드 브랜드 컬러 사용

4. **저장:**
   - PNG 또는 JPEG 형식
   - 1024x500px로 저장
   - 파일 크기 확인 (15MB 이하)

5. **업로드:**
   - Google Play Console에 업로드

---

### 옵션 2: 나중에 추가

1. **현재 상태:**
   - Feature Graphic은 필수이지만, 나중에 추가 가능
   - 앱 아이콘과 스크린샷을 먼저 업로드

2. **작업 순서:**
   - 앱 아이콘 업로드
   - 휴대전화 스크린샷 업로드
   - Feature Graphic은 비워두고 다음 단계로 진행
   - 나중에 디자인해서 추가

3. **장점:**
   - 시간을 두고 신중하게 디자인 가능
   - 앱 출시를 먼저 진행 가능

---

### 옵션 3: 임시로 스크린샷 리사이즈 (비권장)

**주의:** 이 방법은 비율이 맞지 않아 이미지가 잘릴 수 있습니다.

1. **이미지 리사이즈:**
   ```bash
   # 온라인 도구 사용
   https://resizeimage.net/
   # 또는
   https://www.iloveimg.com/resize-image
   ```

2. **설정:**
   - 원본: 800x1244px
   - 타겟: 1024x500px
   - 비율 유지: 해제 (비율이 맞지 않으므로)

3. **문제점:**
   - 이미지가 많이 잘릴 수 있음
   - 화질 저하 가능
   - 전문적인 Feature Graphic이 아님

---

## 💡 추천 작업 순서

### 즉시 진행:

1. ✅ **앱 아이콘 업로드** (이미 완료)
2. ✅ **휴대전화 스크린샷 업로드** (준비됨)
3. ⏳ **Feature Graphic** → 나중에 추가

### 나중에 추가:

4. **Feature Graphic 디자인**
   - Canva 또는 Figma 사용
   - 1024x500px로 디자인
   - 톡보드 브랜드와 주요 기능 포함

5. **업로드**
   - Google Play Console에 추가

## ⚠️ 주의사항

### 이미지 품질:
- 고해상도로 저장 (1024x500px 정확히)
- 텍스트가 읽기 쉬운지 확인
- 다양한 화면 크기에서 테스트

- 너무 많은 정보 포함하지 않기
