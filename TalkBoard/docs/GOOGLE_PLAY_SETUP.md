## 📋 앱 설정 완료 방법

### 방법 1: 좌측 메뉴에서 이동 (권장)

1. **좌측 메뉴에서 "테스트 및 출시"** 클릭
2. 하위 메뉴에서 **"프로덕션"** 또는 **"내부 테스트"** 선택
   - 처음에는 "내부 테스트"부터 시작하는 것이 좋습니다

또는

1. **좌측 메뉴에서 "앱 콘텐츠"** 클릭
   - 앱 정보, 스크린샷 등 설정 가능

### 방법 2: 대시보드에서 직접 이동

1. **"앱 설정 완료"** 섹션에서
2. "앱 정보를 제공하고 스토어 등록정보를 만드세요" 텍스트 클릭
   - 또는 관련 버튼이 있으면 클릭

## 🎯 설정 순서 (권장)

### 1단계: 앱 콘텐츠 입력

**좌측 메뉴** → **"앱 콘텐츠"** 클릭

필수 입력 항목:
- [ ] **앱 이름**: 톡보드
- [ ] **짧은 설명**: 140자 이내
- [ ] **긴 설명**: 4000자 이내
- [ ] **앱 아이콘**: 512x512px PNG (✅ 이미 생성됨)
- [ ] **스크린샷**: 최소 2개 (✅ 이미 준비됨 - 7개)
- [ ] **카테고리**: 커뮤니케이션 또는 소셜
- [ ] **연락처 이메일**: 필수
- [ ] **개인정보처리방침 URL**: 
  - `https://goalsgo1.github.io/talkboard-privacy-policy/`

### 2단계: App Bundle 업로드

**좌측 메뉴** → **"테스트 및 출시"** → **"프로덕션"** (또는 "내부 테스트")

1. **"새 버전 만들기"** 또는 **"프로덕션 출시"** 버튼 클릭
2. **"App Bundle 또는 APK 업로드"** 섹션에서
3. `.aab` 파일 업로드
   - 먼저 빌드 필요: `flutter build appbundle --release`
   - 빌드 파일 위치: `android/app/build/outputs/bundle/release/app-release.aab`

### 3단계: 콘텐츠 등급 설정

**좌측 메뉴** → **"정책"** → **"앱 콘텐츠"**

- 콘텐츠 등급 설문조사 완료
- 일반적으로 "전체 이용가" 선택 가능

## 📝 가장 빠른 시작 방법

### 즉시 시작 가능한 작업:

1. **좌측 메뉴에서 "앱 콘텐츠" 클릭**

## ✅ 체크리스트

### 앱 정보 입력 (우선순위 1)
- [ ] 좌측 메뉴 → "앱 콘텐츠" 클릭
- [ ] 앱 이름 확인/수정: 톡보드
- [ ] 짧은 설명 작성 (140자 이내)
- [ ] 긴 설명 작성 (4000자 이내)
