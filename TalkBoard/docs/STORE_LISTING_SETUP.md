# 스토어 등록정보 설정 가이드

## ✅ 현재 상태

- ✅ 내부 테스트 업로드 완료
- ⏳ 스크린샷 업로드 필요
- ⏳ 개인정보처리방침 URL 입력 필요

---

## 🚀 스크린샷 및 개인정보처리방침 URL 설정하기

### Step 1: "앱 콘텐츠" 메뉴로 이동

1. **좌측 메뉴에서 "앱 콘텐츠" 클릭**
   - 현재 "테스트 및 출시" 페이지에 있습니다
   - 좌측 사이드바에서 "앱 콘텐츠" 찾아서 클릭

### Step 2: "스토어 등록정보" 섹션 찾기

"앱 콘텐츠" 페이지에서:
1. **"스토어 등록정보"** 또는 **"Store Listing"** 섹션 찾기
2. 클릭하여 확장

---

## 📸 스크린샷 업로드

### 위치:
- "앱 콘텐츠" → "스토어 등록정보" → "그래픽 자산" 또는 "스크린샷" 섹션

### 업로드할 파일 (7개):
```
screenshots/android/01_login.png
screenshots/android/02_signup.png
screenshots/android/03_post_list.png
screenshots/android/04_post_create.png
screenshots/android/05_post_detail.png
screenshots/android/06_post_detail_comments.png
screenshots/android/07_post_detail_replies.png
```

### 업로드 방법:
1. **"스크린샷"** 섹션 찾기
2. **"추가"** 또는 **"업로드"** 버튼 클릭
3. 파일 선택 또는 드래그 앤 드롭
4. 최소 2개 이상 업로드 (권장: 3-5개)
5. 업로드 완료 확인

---

## 🔗 개인정보처리방침 URL 입력

### 위치:
- "앱 콘텐츠" → "스토어 등록정보" → "개인정보처리방침" 섹션
- 또는 "앱 콘텐츠" → "개인정보처리방침" 섹션

### 입력할 URL:
```
https://goalsgo1.github.io/talkboard-privacy-policy/
```

### 입력 방법:
1. **"개인정보처리방침"** 섹션 찾기
2. **URL 입력 필드**에 위 URL 입력
3. **"저장"** 버튼 클릭

---

## 📋 전체 설정 체크리스트

### 스토어 등록정보:
- [ ] 앱 이름 확인: 톡보드
- [ ] 짧은 설명 작성 (140자 이내)
- [ ] 긴 설명 작성 (4000자 이내)
- [ ] 스크린샷 업로드 (7개 준비됨)
- [ ] 앱 아이콘 업로드 (512x512px)
- [ ] 카테고리 선택
- [ ] 연락처 이메일 입력
- [ ] 개인정보처리방침 URL 입력
  - URL: `https://goalsgo1.github.io/talkboard-privacy-policy/`

## 💡 빠른 접근 방법

### 방법 1: 좌측 메뉴 사용
1. 좌측 사이드바에서 **"앱 콘텐츠"** 찾기
2. 클릭

### 방법 2: 상단 메뉴 사용
일부 Google Play Console 버전에서는:
1. 상단 메뉴에서 **"앱 콘텐츠"** 찾기
2. 클릭

### 방법 3: 대시보드에서
대시보드 페이지에서:
1. **"앱 설정 완료"** 섹션 찾기
2. **"앱 정보를 제공하고 스토어 등록정보를 만드세요"** 링크 클릭
