# 스토어 등록정보 자산 업로드 가이드

## 📋 업로드 섹션별 가이드

### 1. 앱 아이콘 * (필수)

**요구사항:**
- 형식: PNG 또는 JPEG
- 최대 크기: 1MB
- 해상도: 512 x 512px

**업로드 방법:**
1. 이전에 생성한 `app_icon.png` 파일을 준비합니다
2. "애셋 추가" 버튼을 클릭하거나 파일을 드래그 앤 드롭합니다
3. `app_icon.png` 파일을 선택하여 업로드합니다

**파일 위치:**
- `assets/icon/app_icon.png` (이전에 생성한 파일)

---

### 2. 그래픽 이미지 * (Feature Graphic, 필수)

**요구사항:**
- 형식: PNG 또는 JPEG
- 최대 크기: 15MB
- 해상도: 1024 x 500px (정확히)

**업로드 방법:**
1. **현재 상태:** Feature Graphic 이미지가 아직 준비되지 않았습니다
2. **옵션 1:** Feature Graphic을 생성해야 합니다
   - 톡보드 앱 로고와 메인 기능을 보여주는 이미지
   - 1024 x 500px 크기로 제작 필요
3. **옵션 2:** 나중에 추가하고, 일단 비워두고 다음 단계로 진행
   - 이 항목은 필수이지만, 나중에 추가 가능

**참고:**
- Feature Graphic은 Google Play 스토어 목록 상단에 표시됩니다
- 앱의 주요 기능이나 브랜딩을 보여주는 이미지입니다
- 나중에 디자인해서 추가하거나, 기본 텍스트 이미지를 사용할 수 있습니다

---

### 3. 휴대전화 스크린샷 * (필수)

**요구사항:**
- 개수: 2~8장 (권장: 4장 이상)
- 형식: PNG 또는 JPEG
- 최대 크기: 장당 8MB
- 가로세로 비율: 16:9 또는 9:16
- 해상도: 320px ~ 3840px (프로모션 고려 시 최소 1,080px)

**업로드 방법:**
1. 이전에 캡처한 Android 스크린샷을 준비합니다
2. "애셋 추가" 버튼을 클릭합니다
3. `screenshots/android/` 폴더에서 다음 스크린샷을 선택합니다:
   - 로그인/회원가입 화면
   - 게시글 목록 화면
   - 게시글 상세 화면
   - 댓글/대댓글 화면
   - (선택) 삭제된 게시글 목록 화면

**권장 업로드:**
- 최소 4장 이상 (프로모션 고려)
- 각 면이 최소 1,080px 이상인 스크린샷 사용

---

### 4. 7인치 태블릿 스크린샷 * (필수)

**요구사항:**
- 개수: 최대 8장
- 형식: PNG 또는 JPEG
- 최대 크기: 장당 8MB
- 가로세로 비율: 16:9 또는 9:16
- 해상도: 320px ~ 3840px

**업로드 방법:**
1. 이전에 캡처한 태블릿 스크린샷을 준비합니다
2. "애셋 추가" 버튼을 클릭합니다
3. 태블릿 스크린샷 파일을 선택하여 업로드합니다

**참고:**
- 이전에 안내한 Galaxy Tab 7 Lite 스크린샷이 있다면 사용합니다
- 없으면 휴대전화 스크린샷과 동일한 화면을 사용해도 됩니다

---

### 5. 10인치 태블릿 스크린샷 * (필수)

**요구사항:**
- 개수: 최대 8장
- 형식: PNG 또는 JPEG
- 최대 크기: 장당 8MB
- 가로세로 비율: 16:9 또는 9:16
- 해상도: 1,080px ~ 7,680px

**업로드 방법:**
1. 10인치 태블릿 스크린샷이 있다면 업로드합니다
2. **없으면:** 7인치 태블릿 스크린샷 재사용
3. **또는:** 휴대전화 스크린샷 재사용

**참고:**
- 별표(*)가 있으므로 필수 항목입니다
- 같은 화면을 다른 크기로 업로드해도 됩니다
- Google Play Console이 자동으로 크기 조정합니다

---

- 개수: 4~8장

- 최대 크기: 장당 8MB
- 가로세로 비율: 16:9 또는 9:16
- 해상도: 1,080px ~ 7,680px

**업로드 방법:**
1. Chromebook 스크린샷이 있다면 업로드합니다

**참고:**
- 선택 항목이므로 필수는 아닙니다
- Android 스크린샷을 재사용할 수도 있습니다

## 📝 각 섹션별 업로드 방법

### 공통 업로드 방법:

1. **"애셋 추가" 버튼 클릭:**
   - 화면 중앙의 큰 흰색 상자 안에 있는 "애셋 추가" 버튼을 클릭합니다
   - 파일 탐색기(Finder)가 열립니다

2. **파일 선택:**
   - 준비한 파일을 선택합니다
   - 여러 파일을 동시에 선택 가능합니다

3. **드래그 앤 드롭:**
   - 파일을 직접 흰색 상자로 끌어다 놓을 수도 있습니다

4. **업로드 확인:**
   - 업로드된 파일이 표시됩니다
   - 요구사항에 맞는지 자동 검증됩니다

5. **저장/다음:**
   - 모든 필수 항목 업로드 후 "저장" 또는 "다음" 버튼 클릭
