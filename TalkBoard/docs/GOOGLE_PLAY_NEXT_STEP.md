---

## 📋 "앱 콘텐츠"에서 입력할 것들

### 필수 항목:

1. **스토어 등록정보**
   - 앱 이름: 톡보드 (이미 있을 수도 있음)
   - 짧은 설명: 140자 이내
   - 긴 설명: 4000자 이내

2. **그래픽 자산**
   - 앱 아이콘: 512x512px (이미 준비됨)
   - 스크린샷: 최소 2개 (7개 준비됨)
     - `screenshots/android/01_login.png`
     - `screenshots/android/02_signup.png`
     - `screenshots/android/03_post_list.png`
     - `screenshots/android/04_post_create.png`
     - `screenshots/android/05_post_detail.png`
     - `screenshots/android/06_post_detail_comments.png`
     - `screenshots/android/07_post_detail_replies.png`

3. **앱 카테고리**
   - 커뮤니케이션 또는 소셜 선택

4. **연락처 정보**
   - 이메일 주소: 필수

5. **개인정보처리방침**
   - URL: `https://goalsgo1.github.io/talkboard-privacy-policy/`

---

## 🚀 단계별 진행

### 1단계: 좌측 메뉴에서 "앱 콘텐츠" 클릭

### 2단계: "스토어 등록정보" 섹션으로 이동

### 3단계: 다음 정보 입력:
- 앱 이름: 톡보드
- 짧은 설명: 예) "자유롭게 소통하는 게시판 앱"
- 긴 설명: 앱 기능 상세 설명
- 스크린샷 업로드: 7개 파일 업로드
- 앱 아이콘 업로드: 512x512px
- 카테고리 선택
- 연락처 이메일 입력
- 개인정보처리방침 URL 입력
