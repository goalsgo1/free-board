## ✅ 설문지 답변

### 1. "소셜 또는 커뮤니케이션" 중 선택

**답변: "소셜"** 선택

**이유:**
- 톡보드는 대규모 사용자 그룹과의 소통
- 새로운 사람을 만나고 소통하는 것이 주요 목적
- 콘텐츠를 게시하고 공유하는 플랫폼
- 예시: Facebook, Twitter, Instagram과 유사

---

### 2. "앱에서 사용자의 현재 정확한 실제 위치를 다른 사용자와 공유하나요?"

**답변: "아니요"** 선택

**이유:**
- 톡보드 앱에는 위치 정보 기능이 없습니다
- 사용자 위치를 수집하거나 공유하지 않습니다

### 4. "앱에 사용자 또는 사용자 생성 콘텐츠를 차단하는 기능이 포함되어 있나요?"
