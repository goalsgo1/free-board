# 앱 설정 완료 페이지 가이드

## ✅ 맞는 페이지입니다!

현재 "앱 설정 완료" 페이지에 있습니다. 여기서 두 가지 작업을 진행하세요:

---

## 🔗 1. 개인정보처리방침 URL 설정

### 클릭할 항목:
**"개인정보처리방침 설정 >"**

### 설정 방법:
1. **"개인정보처리방침 설정 >"** 클릭
2. **"개인정보처리방침 URL"** 필드 찾기
3. 다음 URL 입력:
   ```
   https://goalsgo1.github.io/talkboard-privacy-policy/
   ```
4. **"저장"** 버튼 클릭

---

## 📸 2. 스크린샷 업로드

### 클릭할 항목:
**"스토어 등록정보 설정 >"**

### 설정 방법:
1. **"스토어 등록정보 설정 >"** 클릭
2. **"그래픽 자산"** 또는 **"스크린샷"** 섹션 찾기
3. **"추가"** 또는 **"업로드"** 버튼 클릭
4. 다음 파일들 업로드 (7개):
   - `screenshots/android/01_login.png`
   - `screenshots/android/02_signup.png`
   - `screenshots/android/03_post_list.png`
   - `screenshots/android/04_post_create.png`
   - `screenshots/android/05_post_detail.png`
   - `screenshots/android/06_post_detail_comments.png`
   - `screenshots/android/07_post_detail_replies.png`
5. 업로드 완료 확인
