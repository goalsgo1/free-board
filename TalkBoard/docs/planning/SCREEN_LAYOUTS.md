## 4. 추모관 생성/수정 화면 (Memorial Create/Edit Screen)

### 레이아웃 디자인

```

│                                         │

│  ┌─────────────────────────────────┐   │

│  │                                 │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │  [다음 →]                        │   │

│  Step 2: 사진 업로드                     │
│  ┌─────┐ ┌─────┐ ┌─────┐              │

│  └─────┘ └─────┘ └─────┘              │

│  └─────────────────────────────────┘   │
│                                         │
│  Step 3: 동영상                          │
│  ┌─────────────────────────────────┐   │
│  │ YouTube 링크 또는                │   │

│  └─────────────────────────────────┘   │
│                                         │
│  Step 4: 편지 작성                       │
│  ┌─────────────────────────────────┐   │

│                                         │
│  Step 5: 설정                            │
│  ┌─────────────────────────────────┐   │

│  │ 기일: [2024-03-15]               │   │
│  │ 공개/비공개: [토글]              │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │  [저장]                          │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 진입 경로
- **추모관 목록**: "추모관 만들기" 버튼 클릭
- **추모관 목록**: 추모관 카드의 "✏️ 편집" 버튼 클릭

### 사용 가능한 기능
1. **Step 1: 기본 정보**
   - 고인 이름 입력 (필수)
   - 생애 요약 입력 (선택)
   - 다음 버튼 → Step 2 이동

2. **Step 2: 사진 업로드**
   - 사진 추가 버튼 → 이미지 선택
   - 사진 미리보기
   - 사진 삭제 버튼
   - 다음 버튼 → Step 3 이동

3. **Step 3: 동영상**
   - YouTube 링크 입력 또는
   - 동영상 업로드 버튼
   - 다음 버튼 → Step 4 이동

4. **Step 4: 편지 작성**
   - 텍스트 편지 입력 또는
   - 음성 편지 녹음
   - 다음 버튼 → Step 5 이동

5. **Step 5: 설정**
   - 생일 선택
   - 기일 선택
   - 공개/비공개 토글
   - 저장 버튼 → 추모관 저장 → 추모관 상세 화면 이동
