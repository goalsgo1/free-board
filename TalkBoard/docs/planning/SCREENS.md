# 📱 화면 구성 및 UI/UX 상세 기획

## 🎨 전체 화면 구성 (총 9개)

| 화면 이름 | 설명 | 주요 기능 |
|-----------|------|----------|
| 홈 화면 | 추천 추모글, 기념일 알림, 방문 수 요약 | 추천 콘텐츠, 알림, 최근 추모관 |
| 추모관 목록 | 내가 만든 추모관 리스트, 공개/비공개 표시 | 리스트 조회, 검색, 필터, 편집/삭제 |
| 추모관 상세 | 고인 정보, 영원한 시간, 사진/영상/편지/댓글/방문 수 | 콘텐츠 조회, 댓글 작성, 추모금 전달, 책장 보기, 공유, 통계, 백업/내보내기 |
| 추모관 생성/수정 | 사진/영상 업로드, 편지 작성, 공개 설정 | 추모관 생성/수정, 콘텐츠 업로드, 아이템 적용 |
| 기도 요청 | 기도글 등록, 함께 기도 참여, 응답 댓글 | 기도글 작성, 기도 참여, 댓글 |
| 감정 공유 게시판 | 감정 나눔 글 CRUD, 댓글/좋아요 | 글 작성, 조회, 댓글, 좋아요 |
| 마이페이지 | 프로필, 추모금 관리, 프리미엄 구독, 설정 | 추모금 출금, 프리미엄 구독, 아이템 구매, 삭제된 추모관 복원 |
| 공개 추모관 검색 | 공개된 추모관 검색 및 조회 | 검색, 필터, 추모관 조회 |
| 책장 디자인 | 과거와 미래를 볼 수 있는 추억 탐색 | 추억 탐색, 영원한 시간 확인 |

## 1. 홈 화면

### 목적
사용자에게 오늘의 추모 콘텐츠와 기념일 알림을 감성적으로 전달

### 기능
- **추천 추모글 카드**: 최근 등록된 추모관 3개 미리보기
- **오늘 기념일 알림**: 오늘 기념일인 추모관 알림
- **가슴을 울리는 추모글**: 작성자가 공개 동의한 감동적인 추모글 표시
- **최근 등록된 추모관**: 가로 스크롤 카드 형태

### UI 구성

#### 상단 영역
- **앱 로고/제목**: "기억의 정원"
- **알림 아이콘**: 기념일 알림 개수 표시

#### 메인 콘텐츠
1. **앱의 목적과 마인드 섹션**
   - 제목: "기억의 정원" (하트 아이콘과 함께)
   - 메인 메시지: "슬픔은 나눌 때 치유됩니다"
   - 서브 메시지:
     - "함께 공감하고 위로하며, 소중한 기억을 간직하세요."
     - "슬픔을 혼자 간직하기보다, 함께 나누며 위로받으세요."
   - 하단 문구: "함께하는 추모, 함께하는 위로"
   - 디자인:
     - 그라데이션 배경 (베이지 톤)
     - 따뜻한 색감의 카드형 레이아웃
     - 부드러운 그림자 효과
     - 중앙 정렬된 텍스트

2. **기념일 알림 섹션**
   - 카드형 레이아웃
   - 고인 이름 + 날짜 + "추모하러 가기" 버튼
   - 배경: 흐릿한 꽃무늬

3. **추천 추모글 섹션**
   - 제목: "오늘의 추모"
   - 타입별 탭: "사람" / "애완동물" / "전체" 탭으로 구분
   - 카드형 추모글 (3개)
   - 각 카드: 사진 + 편지 요약 + '자세히 보기' 버튼
   - 타입별 아이콘 표시 (사람: 하트, 애완동물: 발자국/고양이/강아지)
   - 가로 스크롤 또는 그리드 레이아웃

4. **가슴을 울리는 추모글 섹션**
   - 제목: "가슴을 울리는 추모글" (하트 아이콘)
   - 타입별 탭: "사람" / "애완동물" / "전체" 탭으로 구분
   - 작성자가 홈 화면 공개에 동의한 추모글 표시
   - 각 카드 구성:
     - 고인 프로필 사진 (원형)
     - 고인 이름 (큰 폰트)
     - 작성자 정보 + 작성 시간
     - 편지 내용 일부 (4줄 미리보기)
     - 하트 아이콘 (감정 표현)
     - "전체 보기" 버튼
     - 공유 버튼
   - 세로 스크롤 리스트 형태
   - 그라데이션 배경 (흰색 → 연한 핑크 → 베이지)

5. **최근 등록된 추모관 섹션**
   - 제목: "최근 등록된 추모관"
   - 가로 스크롤 카드
   - 각 카드: 고인 사진 + 이름 + 기념일

#### 하단 네비게이션
- 홈, 추모관 목록, 기도 요청, 감정 공유, 마이페이지

### 스타일
- **색상**: 따뜻한 톤 (베이지, 크림, 연한 파스텔)
- **레이아웃**: 여백 많은 카드형 구성
- **배경**: 흐릿한 꽃무늬 또는 빛 번짐 효과
- **타이포그래피**: 읽기 쉬운 명조체

## 2. 추모관 목록 화면

### 목적
내가 만든 추모관을 리스트로 확인하고 관리

### 기능
- **추모관 리스트**: 내가 만든 추모관 전체 조회
- **검색/필터**: 
  - 고인 이름으로 검색
  - 추모 대상 타입 필터 (사람/애완동물/전체)
  - 공개/비공개 필터
- **공개/비공개 표시**: 각 추모관의 공개 상태 표시
- **타입별 아이콘 표시**: 사람(하트) / 애완동물(발자국/고양이/강아지)
- **삭제/편집**: 추모관 삭제 및 편집 버튼

### UI 구성

#### 상단 영역
- **제목**: "내 추모관"
- **검색창**: 고인 이름 검색
- **필터 버튼**: 
  - 추모 대상 타입 필터 (사람/애완동물/전체)
  - 공개/비공개 필터

#### 리스트 영역
- **카드형 리스트**: 각 추모관을 카드로 표시
- **카드 구성**:
  - 왼쪽: 고인 사진 (원형 또는 사각형) + 타입별 아이콘
  - 오른쪽: 고인 이름, 기념일, 편지 요약
  - 하단: 공개/비공개 아이콘, 편집/삭제 버튼
- **부드러운 그림자 효과**

#### 하단 버튼
- **추모관 만들기**: 플로팅 액션 버튼 또는 하단 고정 버튼

### 스타일
- **카드 디자인**: 사진 왼쪽, 텍스트 오른쪽
- **그림자**: 부드러운 그림자 효과
- **색상**: 따뜻한 톤 유지

## 3. 추모관 상세 화면

### 목적
고인을 위한 추모 공간을 감성적으로 보여주고, 다양한 콘텐츠로 기억을 나눔

### 기능
- **고인 정보**: 이름, 생애 요약
- **타입별 표현 차이**:
  - **사람 추모관**: "고인", "기일", "돌아가신 날" 표현 사용
  - **애완동물 추모관**: "반려동물", "떠난 날", "무지개 다리를 건넌 날" 표현 사용
- **영원한 시간**: 고인이 돌아가신 날부터 새로운 삶의 시간 시각화
- **사진 갤러리**: 최대 5장 (슬라이드 또는 그리드)
- **동영상**: YouTube 링크 또는 업로드된 영상 재생
- **편지**: 텍스트 또는 음성 재생
- **댓글/위로 메시지**: 작성 및 조회
- **방문 횟수**: "이 추모관은 N명이 방문했습니다"
- **추모금 전달**: 금액 입력 및 결제 (사람 추모관에만 표시 가능)

### UI 구성

#### 상단 영역
- **타입별 아이콘**: 사람(하트) / 애완동물(발자국/고양이/강아지)
- **고인 이름**: 큰 글씨로 표시
- **생애 요약**: 고인의 생애를 간단히 요약
- **공개/비공개 아이콘**: 추모관 공개 상태

#### 영원한 시간 섹션
- **위치**: 상단 영역 바로 아래, 콘텐츠 영역 위
- **디자인**: 
  - 따뜻한 베이지/크림 배경의 카드형 레이아웃
  - 부드러운 시계 아이콘 또는 시간 흐름 애니메이션
  - 실시간으로 업데이트되는 시간 표시
- **표시 내용**:
  - "새로운 삶의 시간: N년 N개월 N일 N시간 N분"
  - "영원한 시간이 흐르고 있습니다" 같은 위로의 메시지
  - 고인이 돌아가신 날짜 표시
- **애니메이션**:
  - 부드럽게 흐르는 시계 바늘 또는 시간 흐름 효과
  - 따뜻한 빛이 흐르는 듯한 효과
- **의미**:
  - 죽음은 끝이 아니라 새로운 삶의 시작
  - 고인이 돌아가신 순간부터 새로운 시간이 흐르기 시작
  - 죽음으로 인해 다시 태어난 것처럼 새로운 삶의 시간을 가지게 됨
  - 이는 위로와 희망을 주는 의미

#### 콘텐츠 영역
1. **사진 갤러리**
   - 슬라이드 또는 그리드 레이아웃
   - 최대 5장 표시
   - 이미지 확대 기능

2. **동영상 영역**
   - 썸네일 + 재생 버튼
   - YouTube 링크 또는 업로드된 영상

3. **편지 영역**
   - 감성적 배경 이미지
   - 텍스트 편지: 마크다운 지원
   - 음성 편지: 재생 버튼 + 진행 바

#### 댓글 영역
- **댓글 작성창**: 텍스트 입력 + 전송 버튼
- **댓글 리스트**: 카드형 댓글
  - 작성자 이름, 작성 시간, 댓글 내용
  - 수정/삭제 버튼 (작성자만)

#### 하단 영역
- **방문 수**: "이 추모관은 37명이 방문했습니다"
- **추모금 전달 버튼**: 금액 입력 모달 열기 (사람 추모관에만 표시)
- **공유 버튼**: 추모관 공유 모달 열기 (공개 추모관만 가능)
- **통계 버튼**: 추모관 통계 화면 이동 (생성자만)
- **백업/내보내기 버튼**: 백업/내보내기 화면 이동 (생성자만)

### 추모금 전달 모달
- **금액 입력**: 숫자 키패드
- **수수료 안내**: "전달 금액의 5% 수수료 및 ₩1,900 유지비가 차감됩니다"
- **결제 수단 선택**: 카드, 계좌이체, 간편결제
- **결제 버튼**: "추모금 전달하기"

### 추모관 공유 모달
- **공유 링크**: 추모관 링크 생성 및 복사
- **QR 코드**: QR 코드 이미지 생성 및 저장
- **SNS 공유**: 카카오톡, 페이스북, 트위터 등으로 공유
- **공유 통계**: 공유 횟수 및 공유를 통해 방문한 사람 수

### 추모관 통계 화면
- **기간 선택**: 전체, 최근 1개월, 3개월, 6개월
- **방문 수 그래프**: 일별/월별 방문 수 그래프
- **댓글 수 통계**: 일별/월별 댓글 수 그래프
- **추모금 통계**: 일별/월별 추모금 수령액 그래프
- **공감 수 통계**: 좋아요 및 댓글 수 통계
- **공유 통계**: 공유 횟수 및 공유를 통해 방문한 사람 수

### 추모관 백업/내보내기 화면
- **PDF 내보내기**: 추모관 전체 내용을 PDF로 저장
- **사진/영상 다운로드**: 모든 사진과 영상을 원본 품질로 다운로드
- **편지 텍스트 내보내기**: 모든 편지 내용을 텍스트 파일로 저장
- **전체 데이터 백업**: 추모관 전체 데이터를 JSON 형식으로 백업

### 스타일
- **사진 중심**: 전시형 레이아웃
- **편지 배경**: 흐릿한 이미지 배경
- **댓글**: 카드형 리스트
- **색상**: 따뜻한 톤 유지

## 4. 추모관 생성/수정 화면

### 목적
고인을 위한 추모관을 만들거나 수정

### 기능
- **고인 정보 입력**: 이름, 생애 요약
- **사진 업로드**: 최대 5장 (프리미엄: 무제한)
- **동영상 업로드**: YouTube 링크 또는 직접 업로드
- **편지 작성**: 텍스트 또는 음성
- **기념일 설정**: 생일, 기일 선택
- **공개/비공개 설정**: 토글 스위치
- **저장**: 추모관 저장

### UI 구성

#### 단계별 입력 (Step 1~6)

**Step 1: 추모 대상 선택**
- 추모 대상 타입 선택 (필수)
  - "사람" 버튼: 전통적인 추모 분위기, "고인", "기일" 표현 사용
  - "애완동물" 버튼: 따뜻하고 밝은 분위기, "반려동물", "떠난 날" 표현 사용
- 각 타입에 맞는 아이콘과 색상으로 시각적 구분
- 다음 버튼

**Step 2: 기본 정보**
- 고인 이름 입력 (필수)
- 생애 요약 입력 (선택)
- 다음 버튼

**Step 3: 사진 업로드**
- 사진 업로드 버튼
- 미리보기 그리드 (최대 5장)
- 삭제 버튼 (각 사진)
- 다음 버튼

**Step 4: 동영상**
- YouTube 링크 입력 또는
- 동영상 업로드 버튼
- 미리보기
- 다음 버튼

**Step 5: 편지 작성**
- 텍스트 편지: 텍스트 입력창 (마크다운 지원)
- 음성 편지: 녹음 버튼 + 재생 버튼
- 홈 화면 공개 설정: 토글 스위치
  - "홈 화면에 공개하여 더 많은 사람들과 나누기" (기본값: 비공개)
  - 공개 선택 시: "가슴을 울리는 추모글"에 표시 가능
  - 비공개 선택 시: 추모관 내에서만 조회 가능
- 다음 버튼

**Step 6: 설정**
- 기념일 선택: 날짜 선택 위젯 (생일, 기일)
- 공개/비공개: 토글 스위치
- 저장 버튼 (하단 고정)

- **카드형 기도글**:
  - 제목
  - 내용
