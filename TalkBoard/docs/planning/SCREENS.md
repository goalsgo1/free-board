# 📱 화면 구성 및 UI/UX 상세 기획

## 📱 반응형 레이아웃 가이드

모든 화면은 다양한 모바일 기기 크기에서도 레이아웃이 깨지지 않도록 반응형으로 설계되어야 합니다.

### 반응형 레이아웃 원칙
- **최소 화면 크기**: 320px × 568px (iPhone SE)
- **텍스트 오버플로우 방지**: 모든 텍스트는 `maxLines`와 `TextOverflow.ellipsis` 사용
- **제목 잘림 방지**: 제목은 최대 2줄까지, 필요시 "..." 표시
- **입력 필드**: 최소 높이 44px, 폰트 크기 최소 16px
- **버튼**: 작은 화면에서는 전체 너비 사용
- **이미지**: 화면 너비를 초과하지 않도록, 비율 유지
- **스크롤**: 모든 화면에서 `SingleChildScrollView` 사용

**자세한 반응형 레이아웃 가이드**: [RESPONSIVE_LAYOUT.md](./RESPONSIVE_LAYOUT.md) 참고

## 1. 홈 화면

### 목적
사용자에게 오늘의 추모 콘텐츠와 기념일 알림을 감성적으로 전달

### 기능
- **추천 추모글 카드**: 최근 등록된 추모관 3개 미리보기
- **오늘 기념일 알림**: 오늘 기념일인 추모관 알림
- **가슴을 울리는 추모글**: 작성자가 공개 동의한 감동적인 추모글 표시
- **애완동물 추모관 안내**: 애완동물과의 추억도 함께 간직할 수 있다는 안내
- **최근 등록된 추모관**: 가로 스크롤 카드 형태
- **홈 내비게이터**: 앱의 모든 주요 기능과 진입 경로를 안내하는 기능 지도

### UI 구성

#### 상단 영역
- **앱 로고/제목**: "기억의 정원"
- **알림 아이콘**: 기념일 알림 개수 표시

#### 메인 콘텐츠
1. **앱의 목적과 마인드 섹션**
   - 제목: "기억의 정원" (하트 아이콘과 함께)
   - 메인 메시지: "슬픔은 나눌 때 치유됩니다"
   - 서브 메시지:
     - "함께 공감하고 위로하며, 소중한 기억을 간직하세요."
     - "슬픔을 혼자 간직하기보다, 함께 나누며 위로받으세요."
   - 하단 문구: "함께하는 추모, 함께하는 위로"
   - 디자인:
     - 그라데이션 배경 (베이지 톤)
     - 따뜻한 색감의 카드형 레이아웃
     - 부드러운 그림자 효과
     - 중앙 정렬된 텍스트

2. **기념일 알림 섹션**
   - 카드형 레이아웃
   - 고인 이름 + 날짜 + "추모하러 가기" 버튼
   - 배경: 흐릿한 꽃무늬

3. **추천 추모글 섹션**
   - 제목: "오늘의 추모"
   - 타입별 탭: "사람" / "애완동물" / "전체" 탭으로 구분
   - 카드형 추모글 (3개)
   - 각 카드: 사진 + 편지 요약 + '자세히 보기' 버튼
   - 타입별 아이콘 표시 (사람: 하트, 애완동물: 발자국/고양이/강아지)
   - 가로 스크롤 또는 그리드 레이아웃

4. **가슴을 울리는 추모글 섹션**
   - 제목: "가슴을 울리는 추모글" (하트 아이콘)
   - 타입별 탭: "사람" / "애완동물" / "전체" 탭으로 구분
   - 작성자가 홈 화면 공개에 동의한 추모글 표시
   - 각 카드 구성:
     - 고인 프로필 사진 (원형)
     - 고인 이름 (큰 폰트)
     - 작성자 정보 + 작성 시간
     - 편지 내용 일부 (4줄 미리보기)
     - 하트 아이콘 (감정 표현)
     - "전체 보기" 버튼
     - 공유 버튼
   - 세로 스크롤 리스트 형태
   - 그라데이션 배경 (흰색 → 연한 핑크 → 베이지)

5. **애완동물 추모관 안내 섹션**
   - 제목: "🐾 애완동물과의 추억도 함께"
   - 메인 메시지: "사랑하는 반려동물도 여기서 기억할 수 있습니다"
   - 애완동물 아이콘: 강아지, 고양이, 햄스터, 토끼 등
   - 설명: "강아지, 고양이, 새, 햄스터, 토끼... 함께했던 모든 순간을 소중히 간직하세요"
   - "애완동물 추모관 만들기" 버튼
   - 디자인:
     - 그라데이션 배경 (연한 초록 톤)
     - 따뜻하고 밝은 색감
     - 부드러운 그림자 효과
     - 애완동물 아이콘 4개 가로 배치

6. **최근 등록된 추모관 섹션**
   - 제목: "최근 등록된 추모관"
   - 가로 스크롤 카드
   - 각 카드: 고인 사진 + 이름 + 기념일

7. **홈 내비게이터 섹션**
   - 제목: "🧭 기억의 탐험 가이드"
   - 설명: "앱의 모든 기능을 한눈에 확인하고 바로 이동할 수 있는 길잡이"
   - 카드 형태로 기능 카테고리별 그룹화
   - 각 기능 카드:
     - 기능 이름과 아이콘
     - 기능 설명
     - 진입 경로 안내
     - "바로 가기" 버튼 (해당 화면으로 이동)
     - "경로 안내" 버튼 (단계별 경로 팝업 표시)
   - 카테고리:
     - 📁 추모관 관리 (추모관 만들기, 내 추모관 보기, 추모관 상세, 영원한 시간, 책장 보기, 추모금 전달)
     - 💬 공감 및 위로 (감정 공유 게시판, 후회 없는 편지, 기도 요청, 댓글 작성)
     - 🤖 AI 기능 (AI 기억 답장)
     - 📜 특별 기능 (유언장, 상호 위로 매칭)
     - 🔍 검색 및 공유 (공개 추모관 검색, 추모관 공유, 추모관 통계)
     - 🎁 감사 혜택 (감사 혜택 보기)
     - ⚙️ 설정 및 관리 (프로필 설정, 알림 설정, 추모금 관리, 프리미엄 구독, 추모관 백업, 전체 설정)
   - 구성:
     - 카테고리별 카드 (추모관 관리, 공감 나눔, AI 기억, 유언장, 매칭, 감사 혜택, 설정 등)
     - 각 카드에 대표 아이콘, 한 줄 설명, "바로 가기" 버튼
   - 상호작용:
     - 버튼 탭 시 해당 화면으로 이동 (예: 추모관 만들기 → 추모관 생성 화면)
     - 선행 조건이 필요한 경우 안내 다이얼로그 표시 (예: 프리미엄 전용 기능)
     - "모든 화면 보기" 버튼으로 UI 프리뷰 모드 진입 가능
   - 디자인:
     - 그리드 또는 2열 카드 레이아웃
     - 각지고 무거운 테두리 + 그림자
     - 카드 배경 색상은 기능 카테고리별로 구분
   - 반응형:
     - 모바일: 세로 스크롤, 두 줄 이상 길어지면 접기/펼치기 제공
     - 태블릿/웹: 3~4열 그리드 레이아웃, 텍스트와 버튼을 가로 배치

#### 하단 네비게이션
- 홈, 추모관 목록, 기도 요청, 감정 공유, 마이페이지

### 스타일
- **색상**: 따뜻한 톤 (베이지, 크림, 연한 파스텔)
- **레이아웃**: 여백 많은 카드형 구성
- **배경**: 흐릿한 꽃무늬 또는 빛 번짐 효과
- **타이포그래피**: 읽기 쉬운 명조체

## 1-1. 홈 내비게이터 (기억의 탐험 가이드)

### 목적
앱의 모든 주요 기능을 한눈에 확인하고, 각 기능으로 바로 이동하거나 경로 안내를 받을 수 있는 통합 길잡이 화면

### 기능
- **기능 카테고리별 분류**: 7개 카테고리로 기능 그룹화
- **기능 상세 정보 표시**: 각 기능의 설명과 진입 경로 안내
- **바로 가기**: 해당 화면으로 즉시 이동
- **경로 안내**: 단계별 진입 경로를 팝업으로 표시
- **검색 기능** (선택적): 기능 이름으로 검색 가능

### UI 구성

#### 상단 안내 영역
- **제목**: "기억의 탐험 가이드"
- **설명**: "기억의 정원에서 제공하는 모든 기능을 한눈에 확인하세요"
- **서브 설명**: "각 기능을 탭하면 해당 화면으로 바로 이동하거나, 경로 안내를 확인할 수 있습니다."

#### 카테고리 섹션
1. **📁 추모관 관리**
   - 추모관 만들기
   - 내 추모관 보기
   - 추모관 상세 보기
   - 영원한 시간
   - 책장 보기
   - 추모금 전달

2. **💬 공감 및 위로**
   - 감정 공유 게시판
   - 후회 없는 편지
   - 기도 요청
   - 댓글 작성

3. **🤖 AI 기능**
   - AI 기억 답장

4. **📜 특별 기능**
   - 유언장
   - 상호 위로 매칭

5. **🔍 검색 및 공유**
   - 공개 추모관 검색
   - 추모관 공유
   - 추모관 통계

6. **🎁 감사 혜택**
   - 감사 혜택 보기

7. **⚙️ 설정 및 관리**
   - 프로필 설정
   - 알림 설정
   - 추모금 관리
   - 프리미엄 구독
   - 추모관 백업
   - 전체 설정

#### 기능 카드 구성
각 기능 카드에는 다음이 포함됩니다:
- **아이콘**: 기능을 나타내는 아이콘 (색상별 구분)
- **기능 이름**: 명확한 기능 이름
- **기능 설명**: 한 줄 설명
- **진입 경로**: "홈 → 하단 네비게이션 [추모관 목록] → [+] 버튼" 형식
- **바로 가기 버튼**: 해당 화면으로 즉시 이동
- **경로 안내 버튼**: 단계별 경로를 팝업으로 표시

### 상호작용
- **바로 가기 버튼**: 해당 화면으로 네비게이션 (조건 확인 후 이동)
- **경로 안내 버튼**: 단계별 경로를 다이얼로그로 표시하고, 각 단계에서 바로 가기 제공
- **카테고리 헤더**: 카테고리별 접기/펼치기 기능 (선택적)

### 진입 경로
- **홈 화면**: "기억의 탐험 가이드" 카드/버튼 클릭
- **마이페이지**: "도움말" 또는 "기능 안내" 버튼
- **하단 네비게이션**: (선택적) 도움말 탭

### 반응형 레이아웃
- **작은 화면 (320px~375px)**:
  - 카드 세로 배치
  - 경로 텍스트 2줄까지, 필요시 "..." 표시
  - 버튼 전체 너비 사용
- **큰 화면 (375px+)**:
  - 카드 가로 배치 가능
  - 경로 텍스트 전체 표시
  - 버튼 적절한 패딩

### 스타일
- **카테고리 헤더**: 그라데이션 배경, 각지고 무거운 테두리
- **기능 카드**: 흰색/베이지 그라데이션 배경, 브라운 테두리, 부드러운 그림자

## 3. 추모관 상세 화면

### 목적
고인을 위한 추모 공간을 감성적으로 보여주고, 다양한 콘텐츠로 기억을 나눔

### 기능
- **고인 정보**: 이름, 생애 요약
- **타입별 표현 차이**:
  - **사람 추모관**: "고인", "기일", "돌아가신 날" 표현 사용
  - **애완동물 추모관**: "반려동물", "떠난 날", "무지개 다리를 건넌 날" 표현 사용
- **영원한 시간**: 고인이 돌아가신 날부터 새로운 삶의 시간 시각화
- **사진 갤러리**: 최대 5장 (슬라이드 또는 그리드)
- **동영상**: YouTube 링크 또는 업로드된 영상 재생
- **편지**: 텍스트 또는 음성 재생
- **댓글/위로 메시지**: 작성 및 조회
- **방문 횟수**: "이 추모관은 N명이 방문했습니다"
- **추모금 전달**: 금액 입력 및 결제 (사람 추모관에만 표시 가능)

### UI 구성

#### 상단 영역
- **타입별 아이콘**: 사람(하트) / 애완동물(발자국/고양이/강아지)
- **고인 이름**: 큰 글씨로 표시
- **생애 요약**: 고인의 생애를 간단히 요약
- **공개/비공개 아이콘**: 추모관 공개 상태

#### 영원한 시간 섹션
- **위치**: 상단 영역 바로 아래, 콘텐츠 영역 위
- **디자인**: 
  - 따뜻한 베이지/크림 배경의 카드형 레이아웃
  - 부드러운 시계 아이콘 또는 시간 흐름 애니메이션
  - 실시간으로 업데이트되는 시간 표시
- **표시 내용**:
  - "새로운 삶의 시간: N년 N개월 N일 N시간 N분"
  - "영원한 시간이 흐르고 있습니다" 같은 위로의 메시지
  - 고인이 돌아가신 날짜 표시
- **애니메이션**:
  - 부드럽게 흐르는 시계 바늘 또는 시간 흐름 효과
  - 따뜻한 빛이 흐르는 듯한 효과
- **의미**:
  - 죽음은 끝이 아니라 새로운 삶의 시작
  - 고인이 돌아가신 순간부터 새로운 시간이 흐르기 시작
  - 죽음으로 인해 다시 태어난 것처럼 새로운 삶의 시간을 가지게 됨
  - 이는 위로와 희망을 주는 의미

#### 콘텐츠 영역
1. **사진 갤러리**
   - 슬라이드 또는 그리드 레이아웃
   - 최대 5장 표시
   - 이미지 확대 기능

2. **동영상 영역**
   - 썸네일 + 재생 버튼
   - YouTube 링크 또는 업로드된 영상

3. **편지 영역**
   - 감성적 배경 이미지
   - 텍스트 편지: 마크다운 지원
   - 음성 편지: 재생 버튼 + 진행 바

#### 댓글 영역
- **댓글 작성창**: 텍스트 입력 + 전송 버튼
- **댓글 리스트**: 카드형 댓글
  - 작성자 이름, 작성 시간, 댓글 내용
  - 수정/삭제 버튼 (작성자만)

#### 하단 영역
- **방문 수**: "이 추모관은 37명이 방문했습니다"
- **추모금 전달 버튼**: 금액 입력 모달 열기 (사람 추모관에만 표시)
- **공유 버튼**: 추모관 공유 모달 열기 (공개 추모관만 가능)
- **통계 버튼**: 추모관 통계 화면 이동 (생성자만)
- **백업/내보내기 버튼**: 백업/내보내기 화면 이동 (생성자만)
- **알림 설정 버튼**: 추모관별 알림 설정 화면 이동 (생성자만)

### 추모금 전달 모달
- **금액 입력**: 숫자 키패드
- **수수료 안내**: "전달 금액의 5% 수수료 및 ₩1,900 유지비가 차감됩니다"
- **결제 수단 선택**: 카드, 계좌이체, 간편결제
- **결제 버튼**: "추모금 전달하기"

### 추모관 공유 모달
- **공유 링크**: 추모관 링크 생성 및 복사
- **QR 코드**: QR 코드 이미지 생성 및 저장
- **SNS 공유**: 카카오톡, 페이스북, 트위터 등으로 공유
- **공유 통계**: 공유 횟수 및 공유를 통해 방문한 사람 수

### 추모관 통계 화면
- **기간 선택**: 전체, 최근 1개월, 3개월, 6개월
- **방문 수 그래프**: 일별/월별 방문 수 그래프
- **댓글 수 통계**: 일별/월별 댓글 수 그래프
- **추모금 통계**: 일별/월별 추모금 수령액 그래프
- **공감 수 통계**: 좋아요 및 댓글 수 통계
- **공유 통계**: 공유 횟수 및 공유를 통해 방문한 사람 수

### 추모관 백업/내보내기 화면

     - 기일 알림 ON/OFF
     - 기념일 1주일 전 알림 ON/OFF
  2. **댓글 알림**
     - 새 댓글 알림 ON/OFF
     - 댓글 답글 알림 ON/OFF
  3. **기도 요청 알림**
     - 기도 요청 알림 ON/OFF
     - 기도 참여 알림 ON/OFF
  4. **추모금 알림**
     - 추모금 전달 알림 ON/OFF
     - 추모금 수령 완료 알림 ON/OFF
  5. **방문 수 증가 알림**
     - 방문 수 마일스톤 알림 ON/OFF (10, 50, 100, 500, 1000명)
  6. **공유 알림**
     - 공유 알림 ON/OFF
  7. **영원한 시간 알림**
     - 주간 알림 ON/OFF (매주 일요일)
     - 월간 알림 ON/OFF (매월 1일)

  8. **재활성화 알림** (유령 회원용)

  9. **홈 화면 공개 제안 알림**

  10. **알림 수신 시간**
      - 알림 수신 시작 시간 (기본: 09:00)
      - 알림 수신 종료 시간 (기본: 22:00)
      - 기념일 알림은 시간 제한 없음 (고정 시간 발송)
- **저장**: 설정 변경 시 즉시 저장
- **권한**: 추모관 생성자만 설정 가능

#### 단계별 입력 (Step 1~6)

**Step 1: 추모 대상 선택**
- 추모 대상 타입 선택 (필수)

- 다음 버튼

**Step 6: 설정**
- 기념일 선택: 날짜 선택 위젯 (생일, 기일)
- 공개/비공개: 토글 스위치

## 7. 후회 없는 편지

### 목적
**고인에 대한 후회를 나중에 다른 사람이 보고 꼭 참고해서 후회 없이 하도록 돕는 것**

- 고인에게 전하지 못한 말, 후회, 조언을 편지 형식으로 공유
- 다른 사용자들이 이를 읽고 자신의 삶에서 후회하지 않도록 참고하고 실천할 수 있도록 돕는 것
- 추모를 넘어 공감과 배움, 그리고 실천의 공간으로 확장
- "그때 더 안아줄 걸 그랬어요. 지금 누군가에게 그 마음을 전하세요." 같은 메시지를 통해 현재 살아있는 사람들에게 행동을 촉구

### 기능
- **편지 작성**: 고인에게 전하지 못한 말, 후회, 조언 등을 텍스트로 작성
- **공개 여부 설정**: 전체 공개 / 익명 공개 / 나만 보기 선택 가능
- **카테고리 선택**: 가족, 친구, 연인, 직장, 신앙 등 관계 유형 선택
- **추모관 연결**: 작성한 추모관에 편지를 연결하여 추모관 내 "후회 없는 편지" 탭에 표시
- **공감 버튼**: 다른 사용자들이 공감할 수 있도록 하트 또는 "도움이 되었어요" 버튼 제공
- **댓글 기능**: 다른 사용자들이 자신의 경험이나 감정을 나눌 수 있음
- **필터 기능**: 관계 유형별, 공감순, 최신순으로 편지 탐색 가능

  - 공감 수: 하트 아이콘 + "도움이 되었어요" 수

## 10. 상호 위로 매칭 시스템 (Mutual Comfort Matching)

### 목적
유사한 상실 경험을 가진 사람들을 매칭하여 서로의 빈자리를 보완하고 위로를 나누는 의미 있는 관계 형성

### 기능
- **프로필 등록**: 상실한 관계 유형, 간단한 자기소개, 매칭 조건 설정
- **매칭 유형**: 
  - 엄마 ↔ 딸
  - 아빠 ↔ 아들
  - 엄마 ↔ 아들
  - 아빠 ↔ 딸
  - 친구 ↔ 친구
  - 형제자매 ↔ 형제자매
- **구인/구직 형식**: 
  - **구인**: 상실한 사람을 찾는 사람 (예: 딸을 잃은 엄마가 딸을 찾음)
  - **구직**: 관계를 찾는 사람 (예: 엄마를 잃은 딸이 엄마 같은 관계를 찾음)
- **매칭 조건 설정**: 나이대, 지역, 성격 유형, 상실 경과 시간
- **매칭 신청**: 관심 있는 프로필에 매칭 신청
- **매칭 수락/거절**: 신청받은 매칭을 검토하여 수락 또는 거절
- **채팅 기능**: 매칭 성사 후 안전한 채팅 공간 제공
- **프로필 검증**: 실명 인증 또는 검증된 프로필 표시 (선택)
- **신고 기능**: 부적절한 행위 신고

### UI 구성

#### 매칭 홈 화면
- **제목**: "상호 위로 매칭"
- **설명**: "유사한 상실 경험을 가진 분들과 서로의 빈자리를 채워주며 위로를 나눠보세요"
- **탭**: 구인 목록 / 구직 목록 / 내 매칭
- **프로필 등록 버튼**: 새 프로필 등록 또는 기존 프로필 수정
- **안전 안내**: 이용 약관 및 안전 수칙 안내

#### 프로필 등록 화면
- **등록 유형 선택**: 구인 / 구직
- **상실한 관계 선택**: 
  - 엄마, 아빠, 딸, 아들, 친구, 형제자매 등
- **간단한 자기소개**: 
  - 이름 (익명 선택 가능)
  - 나이 또는 나이대
  - 지역 (선택)
  - 간단한 소개글
- **매칭 조건 설정**:
  - 찾는 관계 유형 (엄마, 딸 등)
  - 선호 나이대
  - 지역 (선택)
  - 성격 유형 (선택)
- **프로필 사진**: 선택적 (익명 가능)
- **이용 약관 동의**: 필수 동의
- **등록 버튼**: 프로필 등록

#### 매칭 목록 화면
- **필터**: 관계 유형, 나이대, 지역, 최신순/추천순
- **프로필 카드**:
  - 프로필 사진 (또는 기본 이미지)
  - 이름/닉네임
  - 나이대
  - 상실한 관계 (예: "딸을 잃은 엄마")
  - 찾는 관계 (예: "딸을 찾습니다")
  - 간단한 소개 요약
  - 검증 배지 (선택)
  - 매칭 신청 버튼
- **상세 보기**: 프로필 카드 클릭 시 상세 정보 표시

#### 프로필 상세 화면
- **프로필 정보**: 
  - 사진, 이름/닉네임, 나이대, 지역
  - 상실한 관계 및 상실 경과 시간
  - 찾는 관계
  - 자기소개 전문
- **매칭 조건**: 선호 조건 표시
- **검증 상태**: 실명 인증 또는 추모관 연동 검증 여부
- **매칭 신청 버튼**: 매칭 신청하기
- **신고 버튼**: 부적절한 프로필 신고

#### 매칭 신청 화면
- **신청 메시지 작성**: 
  - 간단한 인사말 및 자기소개
  - 왜 매칭을 원하는지 간단히 설명
- **신청 전 확인 사항**: 
  - 이용 약관 동의
  - 개인정보 처리 동의
- **신청 버튼**: 매칭 신청 완료

#### 매칭 관리 화면
- **내가 보낸 신청**: 
  - 신청 상태 (대기 중, 수락됨, 거절됨)
  - 신청 취소 버튼
- **내가 받은 신청**: 
  - 신청자 프로필 미리보기
  - 신청 메시지
  - 수락 / 거절 버튼
- **활성 매칭**: 
  - 매칭된 상대방 프로필
  - 채팅하기 버튼
  - 매칭 종료 버튼

#### 채팅 화면
- **상대방 정보**: 프로필 사진, 이름
- **채팅 메시지**: 텍스트 메시지 주고받기
- **안전 안내**: 개인정보 공유 금지, 부적절한 내용 신고 안내
- **신고 버튼**: 부적절한 대화 신고
- **매칭 종료**: 매칭 종료 옵션

### 스타일
- **따뜻하고 안전한 느낌**: 부드러운 색상, 따뜻한 톤
- **신뢰감**: 검증 배지, 안전 안내 강조
- **프라이버시 보호**: 익명 옵션, 개인정보 보호 강조

### 진입 경로
- **마이페이지**: "상호 위로 매칭" 버튼 클릭
- **홈 화면**: "상호 위로 매칭" 섹션 클릭 (선택)

### 안전 기능
- **익명성 보호**: 실명 공개 여부 선택 가능
- **채팅 모니터링**: 부적절한 내용 자동 감지 (선택)
- **신고 시스템**: 부적절한 사용자 신고 및 차단
- **이용 약관 동의**: 매칭 서비스 이용 시 별도 약관 동의 필수
- **프로필 검증**: 선택적 실명 인증 또는 추모관 연동 검증

### 사용 예시
- **구인**: "딸을 잃은 엄마입니다. 딸의 빈자리가 너무 크네요. 엄마를 잃으신 딸과 서로 위로하며 지내고 싶습니다."
- **구직**: "엄마를 잃은 딸입니다. 엄마의 사랑을 그리워하며, 엄마를 잃으신 어머니와 의미 있는 관계를 나누고 싶습니다."

## 11. 감사 혜택 화면 (Gratitude Benefits Screen)

### 목적
진행 중인 감사 혜택을 안내하고, 사용자가 감사 혜택에 참여하여 혜택을 받을 수 있는 공간 제공

### 기능
- **감사 혜택 목록**: 진행 중인 감사 혜택, 예정된 감사 혜택, 종료된 감사 혜택 구분
- **감사 혜택 상세**: 감사 혜택 상세 정보, 참여 조건, 혜택 안내
- **혜택 참여**: 감사 혜택 참여 버튼 클릭으로 자동 혜택 지급 또는 수동 참여
- **내 혜택 조회**: 받은 혜택 목록 (무료 이용권, 쿠폰 등)
- **감사 혜택 알림**: 새로운 감사 혜택 및 종료 임박 알림

### UI 구성

#### 감사 혜택 목록 화면
- **진행 중인 감사 혜택**: 현재 참여 가능한 감사 혜택 목록
- **예정된 감사 혜택**: 곧 시작될 감사 혜택 미리보기
- **내 혜택**: 받은 혜택 목록 및 사용 가능한 쿠폰
- **감사 혜택 배너**: 홈 화면에 표시될 주요 감사 혜택 배너

#### 감사 혜택 상세 화면
- **감사 혜택 제목 및 기간**: 혜택명, 시작일/종료일
- **감사 혜택 설명**: 감사 혜택 상세 내용 및 참여 방법
- **참여 조건**: 참여 가능한 조건 안내
- **혜택 안내**: 받을 수 있는 혜택 상세 설명
- **참여하기 버튼**: 감사 혜택 참여 버튼
- **혜택 받기 버튼**: 조건 충족 시 혜택 받기 버튼

### 스타일
- **따뜻하고 진지한 느낌**: 감사 혜택 배너는 따뜻한 색상
- **혜택 강조**: 무료 이용권, 쿠폰 등 혜택을 시각적으로 강조
- **참여 유도**: 참여하기 쉽도록 명확한 CTA 버튼

### 진입 경로
- **홈 화면**: 감사 혜택 배너 클릭
- **마이페이지**: "감사 혜택" 버튼 클릭
- **하단 네비게이션**: 감사 혜택 탭 (선택)

### 감사 혜택 예시
1. **초기 가입자 감사 혜택**
   - 조건: 앱 출시 후 1000명 이내 가입
   - 혜택: 프리미엄 3개월 무료
   
2. **공감 많이 받은 사용자 감사**
   - 조건: 추모관 공감 수 100개 이상
   - 혜택: 프리미엄 1개월 무료
   
3. **첫 추모관 생성 감사**
   - 조건: 첫 번째 추모관 생성
   - 혜택: 프리미엄 1개월 무료
   
4. **추천인 감사**
   - 조건: 친구 초대 후 친구 가입
   - 혜택: 양쪽 모두 프리미엄 1개월 무료
   
5. **매일 방문 감사**
   - 조건: 연속 7일 로그인
   - 혜택: 프리미엄 3일 무료
   
6. **리뷰 작성 감사**
   - 조건: 앱 스토어 리뷰 작성
   - 혜택: 프리미엄 1주일 무료
