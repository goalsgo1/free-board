## 1. 홈 화면

### 목적
사용자에게 오늘의 추모 콘텐츠와 기념일 알림을 감성적으로 전달

### 기능
- **추천 추모글 카드**: 최근 등록된 추모관 3개 미리보기
- **오늘 기념일 알림**: 오늘 기념일인 추모관 알림
- **최근 등록된 추모관**: 가로 스크롤 카드 형태

### UI 구성

#### 상단 영역
- **앱 로고/제목**: "기억의 정원"
- **알림 아이콘**: 기념일 알림 개수 표시

#### 메인 콘텐츠
1. **기념일 알림 섹션**
   - 카드형 레이아웃
   - 고인 이름 + 날짜 + "추모하러 가기" 버튼
   - 배경: 흐릿한 꽃무늬

2. **추천 추모글 섹션**
   - 제목: "오늘의 추모"
   - 카드형 추모글 (3개)
   - 각 카드: 사진 + 편지 요약 + '자세히 보기' 버튼
   - 가로 스크롤 또는 그리드 레이아웃

3. **최근 등록된 추모관 섹션**
   - 제목: "최근 등록된 추모관"
   - 가로 스크롤 카드
   - 각 카드: 고인 사진 + 이름 + 기념일

#### 하단 네비게이션
- 홈, 추모관 목록, 기도 요청, 감정 공유, 마이페이지

### 스타일
- **색상**: 따뜻한 톤 (베이지, 크림, 연한 파스텔)
- **레이아웃**: 여백 많은 카드형 구성
- **배경**: 흐릿한 꽃무늬 또는 빛 번짐 효과
- **타이포그래피**: 읽기 쉬운 명조체

## 2. 추모관 목록 화면

### 목적
내가 만든 추모관을 리스트로 확인하고 관리

### 기능
- **추모관 리스트**: 내가 만든 추모관 전체 조회
- **검색/필터**: 고인 이름으로 검색, 공개/비공개 필터
- **공개/비공개 표시**: 각 추모관의 공개 상태 표시
- **삭제/편집**: 추모관 삭제 및 편집 버튼

### UI 구성

#### 상단 영역
- **제목**: "내 추모관"
- **검색창**: 고인 이름 검색
- **필터 버튼**: 공개/비공개 필터

#### 리스트 영역
- **카드형 리스트**: 각 추모관을 카드로 표시
- **카드 구성**:
  - 왼쪽: 고인 사진 (원형 또는 사각형)
  - 오른쪽: 고인 이름, 기념일, 편지 요약
  - 하단: 공개/비공개 아이콘, 편집/삭제 버튼
- **부드러운 그림자 효과**

#### 하단 버튼
- **추모관 만들기**: 플로팅 액션 버튼 또는 하단 고정 버튼

### 스타일
- **카드 디자인**: 사진 왼쪽, 텍스트 오른쪽
- **그림자**: 부드러운 그림자 효과
- **색상**: 따뜻한 톤 유지

## 3. 추모관 상세 화면

### 목적
고인을 위한 추모 공간을 감성적으로 보여주고, 다양한 콘텐츠로 기억을 나눔

### 기능
- **고인 정보**: 이름, 생애 요약
- **사진 갤러리**: 최대 5장 (슬라이드 또는 그리드)
- **동영상**: YouTube 링크 또는 업로드된 영상 재생
- **편지**: 텍스트 또는 음성 재생
- **댓글/위로 메시지**: 작성 및 조회
- **방문 횟수**: "이 추모관은 N명이 방문했습니다"
- **추모금 전달**: 금액 입력 및 결제

### UI 구성

#### 상단 영역
- **고인 이름**: 큰 글씨로 표시
- **생애 요약**: 고인의 생애를 간단히 요약
- **공개/비공개 아이콘**: 추모관 공개 상태

#### 콘텐츠 영역
1. **사진 갤러리**
   - 슬라이드 또는 그리드 레이아웃
   - 최대 5장 표시
   - 이미지 확대 기능

2. **동영상 영역**
   - 썸네일 + 재생 버튼
   - YouTube 링크 또는 업로드된 영상

3. **편지 영역**
   - 감성적 배경 이미지
   - 텍스트 편지: 마크다운 지원
   - 음성 편지: 재생 버튼 + 진행 바

#### 댓글 영역
- **댓글 작성창**: 텍스트 입력 + 전송 버튼
- **댓글 리스트**: 카드형 댓글
  - 작성자 이름, 작성 시간, 댓글 내용
  - 수정/삭제 버튼 (작성자만)

#### 하단 영역
- **방문 수**: "이 추모관은 37명이 방문했습니다"
- **추모금 전달 버튼**: 금액 입력 모달 열기

### 추모금 전달 모달
- **금액 입력**: 숫자 키패드
- **수수료 안내**: "전달 금액의 5% 수수료 및 ₩1,900 유지비가 차감됩니다"
- **결제 수단 선택**: 카드, 계좌이체, 간편결제
- **결제 버튼**: "추모금 전달하기"

### 스타일
- **사진 중심**: 전시형 레이아웃
- **편지 배경**: 흐릿한 이미지 배경
- **댓글**: 카드형 리스트
- **색상**: 따뜻한 톤 유지

## 4. 추모관 생성/수정 화면

### 목적
고인을 위한 추모관을 만들거나 수정

### 기능
- **고인 정보 입력**: 이름, 생애 요약
- **사진 업로드**: 최대 5장 (프리미엄: 무제한)
- **동영상 업로드**: YouTube 링크 또는 직접 업로드
- **편지 작성**: 텍스트 또는 음성
- **기념일 설정**: 생일, 기일 선택
- **공개/비공개 설정**: 토글 스위치
- **저장**: 추모관 저장

### UI 구성

#### 단계별 입력 (Step 1~5)

**Step 1: 기본 정보**
- 고인 이름 입력 (필수)
- 생애 요약 입력 (선택)
- 다음 버튼

**Step 2: 사진 업로드**
- 사진 업로드 버튼
- 미리보기 그리드 (최대 5장)
- 삭제 버튼 (각 사진)
- 다음 버튼

**Step 3: 동영상**
- YouTube 링크 입력 또는
- 동영상 업로드 버튼
- 미리보기
- 다음 버튼

**Step 4: 편지 작성**
- 텍스트 편지: 텍스트 입력창 (마크다운 지원)
- 음성 편지: 녹음 버튼 + 재생 버튼
- 다음 버튼

**Step 5: 설정**
- 기념일 선택: 날짜 선택 위젯 (생일, 기일)
- 공개/비공개: 토글 스위치
- 저장 버튼 (하단 고정)

### 스타일
- **단계 표시**: 상단에 진행 바
- **이미지 미리보기**: 강조된 미리보기
- **입력 필드**: 깔끔한 텍스트 필드
- **버튼**: 하단 고정 또는 플로팅

## 5. 기도 요청 화면

### 목적
기도 요청을 등록하고, 공동체가 함께 기도

### 기능
- **기도글 작성**: 제목, 내용 입력
- **함께 기도 버튼**: 클릭 시 카운트 증가
- **응답 댓글**: 기도 응답 메시지 작성
- **기도 완료 체크**: 기도 완료 표시

### UI 구성

#### 상단 영역
- **제목**: "기도 요청"
- **작성 버튼**: 기도글 작성 버튼

#### 기도글 리스트
- **카드형 기도글**:
  - 제목
  - 내용
  - 작성자 이름
  - 작성 시간
  - 함께 기도 카운트: "N명이 함께 기도하고 있습니다"
  - 함께 기도 버튼
  - 기도 완료 체크박스

#### 댓글 영역
- **댓글 작성창**: 텍스트 입력 + 전송 버튼
- **댓글 리스트**: 말풍선 스타일
  - 작성자 이름, 작성 시간, 댓글 내용

### 스타일
- **카드형 리스트**: 깔끔한 카드 디자인
- **댓글**: 말풍선 스타일
- **배경**: 연한 십자가 또는 빛 이미지
- **색상**: 따뜻한 톤 유지

## 6. 감정 공유 게시판

### 목적
슬픔, 기억, 위로의 감정을 나눌 수 있는 공간

### 기능
- **감정 글 작성**: 제목, 내용 입력
- **댓글/좋아요**: 댓글 작성 및 좋아요 표시
- **인기글/추천글**: 분류 및 필터

### UI 구성

#### 상단 영역
- **제목**: "감정 공유"
- **필터**: 인기글, 최신글, 추천글
- **작성 버튼**: 글 작성 버튼

#### 글 리스트
- **카드형 글**:
  - 제목
  - 내용 요약
  - 작성자 이름
  - 작성 시간
  - 좋아요 수: 하트 아이콘 + 숫자
  - 댓글 수: 댓글 아이콘 + 숫자

#### 글 상세
- **제목**: 큰 글씨
- **내용**: 전체 내용 표시
- **좋아요 버튼**: 하트 아이콘
- **댓글 영역**: 댓글 작성창 + 댓글 리스트

#### 플로팅 액션 버튼
- **글 작성**: 플로팅 액션 버튼

### 스타일
- **카드형 리스트**: 감성적 배경
- **좋아요**: 하트 아이콘 (빨간색)
- **색상**: 따뜻한 톤 유지

## 🎨 공통 UI 컴포넌트

### 버튼
- **주요 버튼**: 파란색 배경, 흰색 텍스트
- **보조 버튼**: 투명 배경, 파란색 텍스트
- **위험 버튼**: 빨간색 배경 (삭제 등)

### 입력 필드
- **텍스트 필드**: 둥근 모서리, 부드러운 그림자
- **라벨**: 위에 표시
- **에러 메시지**: 빨간색 텍스트

### 카드
- **기본 카드**: 흰색 배경, 부드러운 그림자
- **강조 카드**: 배경색 또는 테두리

### 네비게이션
- **하단 네비게이션 바**: 5개 아이콘
- **상단 앱 바**: 제목, 액션 버튼
