# 📖 앱 개요

## 🎯 앱의 목적

### 핵심 가치
- 고인을 기억하고 추모할 수 있는 **디지털 공간** 제공
- 사진, 영상, 편지, 기도, 추모금 등 다양한 방식으로 **감정을 표현**
- 유족과 지인들이 함께 추모하며 **공감과 위로의 공동체** 형성

### 타겟 사용자
- **주 사용자**: 고인을 추모하고 싶은 유족 및 지인
- **부 사용자**: 추모 공간을 방문하고 기도/추모금을 전달하는 방문자

### 핵심 차별점
1. **다양한 추모 방식**: 사진, 영상, 편지, 기도, 추모금 통합 제공
2. **감성적 UI/UX**: 따뜻한 톤과 여백 많은 구성으로 감성적 경험 제공
3. **공동체 형성**: 댓글, 기도 참여, 추모금 전달을 통한 공동체 형성
4. **지속 가능한 운영**: 프리미엄 구독 및 추모금 수수료를 통한 수익 모델

## 🏷️ 앱 정보

- **앱 이름**: 기억의 정원 (Memory Garden)
- **카테고리**: 라이프스타일 / 추모
- **플랫폼**: iOS, Android, Web
- **언어**: 한국어 (향후 다국어 지원 가능)

## 📱 주요 화면 (총 6개)

1. **홈 화면** - 추천 추모글, 기념일 알림, 방문 수 요약
2. **추모관 목록** - 내가 만든 추모관 리스트, 공개/비공개 표시
3. **추모관 상세** - 고인 정보, 사진/영상/편지/댓글/방문 수
4. **추모관 생성/수정** - 사진/영상 업로드, 편지 작성, 공개 설정
5. **기도 요청** - 기도글 등록, 함께 기도 참여, 응답 댓글
6. **감정 공유 게시판** - 감정 나눔 글 CRUD, 댓글/좋아요

## 🎨 디자인 컨셉

- **색상**: 따뜻한 톤 (베이지, 크림, 연한 파스텔)
- **레이아웃**: 여백 많은 카드형 구성
- **배경**: 흐릿한 꽃무늬 또는 빛 번짐 효과
- **타이포그래피**: 읽기 쉬운 명조체, 감성적인 서체

## 🔄 개발 단계

1. **Phase 1**: 기본 추모관 생성/조회 기능
2. **Phase 2**: 사진/영상 업로드, 편지 작성
3. **Phase 3**: 댓글, 기도 요청, 감정 공유 게시판
4. **Phase 4**: 추모금 결제 시스템 연동
5. **Phase 5**: 프리미엄 기능, 푸시 알림
6. **Phase 6**: B2B 제휴 기능
