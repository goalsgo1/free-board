## 🎯 앱의 목적

고인을 기억하고 추모할 수 있는 **디지털 공간**을 제공하여:
- 사진, 영상, 편지, 기도, 추모금 등 다양한 방식으로 **감정을 표현**
- 유족과 지인들이 함께 추모하며 **공감과 위로의 공동체**를 형성
- **프리미엄 기능과 추모금 수수료**를 통해 지속 가능한 서비스 운영
