## 🚀 다음 단계: 앱 만들기

### Step 1: "앱 만들기" 버튼 클릭

1. 화면 우측 상단의 **"앱 만들기"** 버튼 클릭
   - 위치: 앱 목록 위, 검색창 오른쪽

### Step 2: 앱 정보 입력

**필수 정보:**
1. **앱 이름**: `톡보드` (또는 원하는 이름)
   - 최대 50자
   - 나중에 수정 가능

2. **기본 언어**: 한국어 (한국어) 선택
   - 앱 설명 등 기본 언어

3. **앱 또는 게임**: 앱 선택
   - 게임이 아닌 일반 앱이므로 "앱" 선택

4. **무료 또는 유료**: 무료 선택
   - 현재는 무료 앱으로 출시

5. **개인정보처리방침**: 체크박스 선택
   - "개인정보처리방침 제공됨" 체크
   - URL 입력: `https://goalsgo1.github.io/talkboard-privacy-policy/`

### Step 3: 앱 등록

1. 모든 필수 정보 입력 후 **"앱 만들기"** 버튼 클릭
2. 앱 생성 완료

   - 앱 콘텐츠 등급 입력
