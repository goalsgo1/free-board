톡보드 - 자유롭게 소통하는 게시판

톡보드는 Firebase를 기반으로 한 실시간 게시판 애플리케이션입니다. 
간편하고 직관적인 인터페이스로 자유롭게 게시글을 작성하고 소통할 수 있습니다.

📝 주요 기능

• 게시글 작성, 수정, 삭제
  - 간단한 제목과 내용으로 게시글 작성
  - 작성한 게시글 수정 및 삭제 가능
  - 실시간으로 게시글 목록 업데이트

• 댓글 및 대댓글
  - 게시글에 댓글 작성
  - 댓글에 대댓글 작성 가능
  - 계층적 댓글 구조로 쉽게 따라갈 수 있습니다

• 실시간 조회수
  - 게시글 조회수 실시간 업데이트
  - 인기 게시글 쉽게 확인

• 수정 히스토리
  - 게시글 및 댓글 수정 이력 관리
  - 언제 수정되었는지 명확하게 표시

• 소프트 삭제 및 복원
  - 실수로 삭제한 게시글 복원 가능
  - 삭제된 게시글 관리 기능

• 사용자 인증
  - 이메일/비밀번호로 안전한 로그인
  - 자신이 작성한 게시글만 수정/삭제 가능

🎨 특징

• Material Design 3 기반의 현대적인 UI
• 실시간 데이터 동기화로 항상 최신 정보 제공
• 직관적이고 사용하기 쉬운 인터페이스
• 안전한 Firebase 기반 인증 시스템

💡 사용 방법

1. 회원가입 또는 로그인
2. 게시글 작성 또는 열람
3. 댓글 작성 및 소통
4. 실시간 업데이트된 정보 확인

🔒 보안 및 개인정보

• Firebase Authentication을 통한 안전한 인증
• 개인정보처리방침 제공
• 사용자 데이터 안전 보관
