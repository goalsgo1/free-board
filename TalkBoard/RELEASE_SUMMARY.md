## 📋 다음 단계 (사용자 작업 필요)

### 즉시 수행 가능

#### 1. Android 서명 키 생성 (약 5분)
```bash
cd /Users/haemin/Documents/GitHub/project/TalkBoard/android/app

keytool -genkey -v -keystore key.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias upload
```

그 다음 `android/key.properties` 파일 생성:
```properties
storePassword=키스토어_비밀번호
keyPassword=키_비밀번호
keyAlias=upload
storeFile=app/key.jks
```

#### 2. 앱 아이콘 커스터마이징 (약 1시간)
- [AppIcon.co](https://www.appicon.co/) 또는 유사 도구 사용
- 1024x1024px 아이콘 이미지 준비
- 생성된 아이콘을 프로젝트에 적용

#### 3. 스크린샷 촬영 (약 30분)
- 앱 실행 후 주요 화면 스크린샷 촬영
- 최소 2개 (Android), 1개 (iOS)
- 권장: 5개 이상 (로그인, 목록, 상세, 댓글, 작성)

#### 4. 앱 스토어 설명 작성 (약 1시간)
- 짧은 설명 (140자 이내)
- 긴 설명 (4000자 이내)
- 키워드 (iOS용)
